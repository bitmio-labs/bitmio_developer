# Create an App Project

App Projects are the containers for your Actions and allow you to manage user access.

Create a new app project:

``` bash
bitmio projects create myapp
```

## List your App Projects

To view all your existing app projects:

``` bash
bitmio projects
```

## Invite App Project collaborators

Invite team members to your App Project:

``` bash
bitmio projects myapp members invite
```

Send the returned invite link to your team.
