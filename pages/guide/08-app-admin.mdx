# Administering the App

App users with the Admin role can invoke any of the admin Actions.

## Viewing Usage
To view your team's usage of Actions:

``` bash
bitmio myapp usage
```

## Inviting Users
To invite new app users:

``` bash
bitmio myapp members invite
```

``` bash
bitmio myapp members invite --email "mail@mirkokiefer.com"
```
